# 🚀 البداية السريعة

## ⚠️ مهم جداً: إعداد Supabase

لكي يعمل التطبيق بشكل كامل، يجب عليك:

### 1️⃣ إنشاء مشروع Supabase (مجاني)

1. اذهب إلى https://supabase.com
2. قم بإنشاء حساب مجاني
3. أنشئ مشروع جديد (New Project)
4. احفظ هذه المعلومات من لوحة التحكم:
   - **Project URL** (من Settings > API)
   - **Anon/Public Key** (من Settings > API)

### 2️⃣ إضافة المعلومات إلى التطبيق

افتح ملف `.env` وأضف المعلومات:

```env
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

### 3️⃣ تفعيل تسجيل الدخول عبر Google (اختياري)

إذا أردت تفعيل Google Sign-In:
1. في Supabase: اذهب إلى **Authentication > Providers**
2. فعّل **Google**
3. اتبع التعليمات في: https://supabase.com/docs/guides/auth/social-login/auth-google

## 📱 الميزات المتاحة

### ✅ بعد تسجيل الدخول:

1. **📊 لوحة التحكم**: تتبع دخلك ومصروفاتك اليومية
2. **📈 محاكي الاستثمار**: تدرب على الاستثمار بأموال افتراضية
3. **🛍️ المنتجات المالية**: تصفح الأسهم والصناديق الاستثمارية
4. **👥 المجتمع**: شارك تجاربك وتنافس مع الآخرين
5. **🏆 الإنجازات**: اجمع النقاط واكسب الشارات
6. **👤 البروفايل**: إدارة معلوماتك الشخصية

---

## 🔐 طرق تسجيل الدخول

### 1. البريد الإلكتروني وكلمة المرور
- اضغط على "تسجيل الدخول" في الأعلى
- اختر "حساب جديد"
- أدخل اسمك، بريدك الإلكتروني، وكلمة مرور
- ابدأ الاستخدام!

### 2. Google Sign-In (يتطلب إعداد)
- بعد إعداد Google Provider في Supabase
- اضغط على "تسجيل الدخول عبر Google"
- اختر حسابك في Google

---

## 🆘 المشاكل الشائعة

### ❌ لا أستطيع تسجيل الدخول
✅ **الحل**: تأكد من إضافة معلومات Supabase في ملف `.env`

### ❌ تسجيل الدخول عبر Google لا يعمل  
✅ **الحل**: يجب تفعيل Google Provider في Supabase أولاً

### ❌ المنتجات المالية لا تظهر
✅ **الحل**: عند أول زيارة، انتظر قليلاً حتى يتم تحميل البيانات تلقائياً
